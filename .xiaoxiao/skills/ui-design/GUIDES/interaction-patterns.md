# 交互模式指南 | Interaction Patterns

## 核心目标

提供常见的用户交互模式，确保一致性和易用性。

## 表单交互

### 实时验证

```markdown
1. 用户输入 → 即时验证 → 显示结果
   ✓ 格式正确 / ✗ 格式错误

2. 错误提示在输入框下方
3. 第一个错误时即提示，不等提交
```

### 必填标记

```markdown
✓ 必填项：标签后加 *
✓ 选填项：标签后加（选填）
```

### 提交状态

```markdown
点击前：[提交按钮]
点击中：[提交中...]
点击后：[提交成功 ✓] → 跳转

错误时：[保持原位，显示错误信息]
```

## 导航交互

### 页面跳转

```markdown
点击链接 → 跳转前页面加载 → 新页面加载完成
         ↓
    加载状态显示
```

### Tab 切换

```markdown
[Tab1] [Tab2] [Tab3]
───────────────────
内容区域1
```

### 下拉菜单

```markdown
点击触发 → 显示菜单 → 选择 → 关闭菜单
                    ↓
              点击外部也关闭
```

## 列表交互

### 加载更多

```markdown
列表内容...
列表内容...
列表内容...
      ↓ 滚动到底部
[加载更多...]
```

### 下拉刷新

```markdown
↓ 继续下拉刷新...
↓
[释放以刷新...]
↓
[刷新中...]
↓
✓ 刷新完成
```

### 批量操作

```markdown
☐ 选择1
☐ 选择2
☐ 选择3
──────────
[批量删除] [批量移动]
（选中项>0时显示）
```

## 反馈交互

### Toast 提示

| 类型 | 用途 | 显示时长 |
|------|------|----------|
| 成功 | 操作成功 | 2秒 |
| 错误 | 操作失败 | 5秒 |
| 警告 | 需要注意 | 3秒 |
| 加载 | 异步操作中 | 直到完成 |

### Modal 弹窗

```markdown
显示弹窗 → 用户操作 → 关闭弹窗
              ↓
         确认/取消
```

### 确认对话框

```markdown
┌─────────────────────────────┐
│ 确认删除？                   │
│                             │
│ 删除后无法恢复              │
│                             │
│    [取消]      [删除]       │
└─────────────────────────────┘
```

## 手势交互

| 手势 | 操作 | 场景 |
|------|------|------|
| 点击 | 选择/触发 | 主要操作 |
| 长按 | 显示更多 | 右键菜单 |
| 滑动 | 快速操作 | 列表项操作 |
| 拖拽 | 移动排序 | 拖放排序 |
| 捏合 | 缩放 | 图片/地图 |

## 键盘交互

### 快捷键

| 快捷键 | 操作 |
|--------|------|
| Enter | 确认/提交 |
| Escape | 取消/关闭 |
| Tab | 下一个输入框 |
| Shift+Tab | 上一个输入框 |

### 焦点管理

```markdown
打开弹窗 → 焦点移到弹窗内
关闭弹窗 → 焦点回到触发元素
```

## 加载状态

### 全局加载

```markdown
[请求时显示loading遮罩]

透明遮罩 + 中心spinner
点击无效
```

### 局部加载

```markdown
[按钮内spinner]
[区域占位+spinner]
[内容区域骨架屏]
```

## 错误处理

### 表单错误

```markdown
1. 输入时实时验证
2. 提交时检查全部
3. 错误信息在字段下方
4. 第一个错误字段获得焦点
```

### 网络错误

```markdown
请求失败
   ↓
显示错误提示 + 重试按钮
   ↓
用户可选择重试或取消
```

## 何时退出

- 所有核心交互已定义
- 异常情况已处理
- 用户体验流程顺畅
- 开发可照图实现
