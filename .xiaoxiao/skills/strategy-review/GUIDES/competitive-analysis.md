# 竞品分析方法 | Competitive Analysis

## 核心目标

了解竞争格局，找到差异化切入点，评估竞争壁垒。

## 竞品分类

| 类型 | 定义 | 示例 |
|------|------|------|
| **直接竞品** | 解决同样问题 | 我们的产品 vs 钉钉 |
| **间接竞品** | 不同解法同一问题 | 我们的产品 vs 微信群 |
| **替代品** | 用户不用任何产品 | 自建 Excel |

## 竞品分析维度

### 1. 产品维度

```
功能覆盖 - 解决了哪些需求
用户体验 - 易用性、学习成本
技术实力 - 性能、稳定性、安全性
```

### 2. 商业维度

```
定价策略 - 免费/订阅/按量
商业模式 - B2B/B2C/B2B2C
销售渠道 - 直销/代理/生态
```

### 3. 市场维度

```
市场份额 - 市场占有率
用户口碑 - NPS、评价
品牌认知 - 品牌知名度
```

## 差异化分析

### 1. 找到空白

```
用户痛点 → 现有竞品哪里满足不了 → 我们能做什么
```

### 2. 评估壁垒

| 壁垒类型 | 说明 | 强度 |
|----------|------|------|
| 技术壁垒 | 专利、专有技术 | 高 |
| 网络效应 | 用户越多越有价值 | 高 |
| 规模效应 | 规模大成本低 | 中 |
| 品牌壁垒 | 品牌认知度 | 中 |
| 转换成本 | 用户换产品代价 | 高 |

### 3. 竞争优势

```
我们有而竞品没有 → 为什么我们能做
竞品有而我们有 → 我们如何做得更好
用户真正在乎 → 我们能否持续投入
```

## 何时退出

- 主要竞品已清晰定位
- 差异化切入点已明确
- 用户确认我们的竞争优势
- 壁垒评估已完成
