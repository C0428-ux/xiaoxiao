# 市场分析方法 | Market Analysis

## 核心目标

量化市场机会，判断产品是否有足够的市场空间支撑发展。

## 市场分析框架

### TAM / SAM / SOM

| 层级 | 定义 | 用途 |
|------|------|------|
| **TAM** | Total Addressable Market - 理论最大市场 | 评估天花板 |
| **SAM** | Serviceable Addressable Market - 可服务市场 | 现实目标 |
| **SOM** | Serviceable Obtainable Market - 可获得市场 | 短期可达 |

### 计算示例

```
TAM:  全球客服软件市场 = $50B
SAM:  中小企业客服市场 = $15B
SOM:  我们第一年能拿到的 = $50M
```

## 市场分析方法

### 1. 需求验证法

```
用户痛点强烈程度 → 愿意付费的用户比例 → 市场大小
```

### 2. 竞品反推法

```
竞品收入 × 竞品市场份额 = 市场总量
```

### 3. 自上而下法

```
行业整体规模 × 目标细分比例 = 目标市场
```

## 市场趋势判断

| 趋势 | 信号 | 影响 |
|------|------|------|
| 增长 | 行业报告、融资事件、招聘增长 | 机会窗口 |
| 稳定 | 成熟市场、无大波动 | 差异化难 |
| 下降 | 客户流失、市场萎缩 | 谨慎进入 |

## 关键问题

- 这个市场在增长还是萎缩？
- 增长/萎缩的速度是多少？
- 驱动增长的因素是什么？
- 这些因素可持续吗？

## 何时退出

- 用户确认市场机会判断
- 有明确的切入点和差异化方向
