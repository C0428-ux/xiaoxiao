## 6. Developer Tools / GitHub Style

**Applicable**: DevTools, IDE extensions, code editors, CI/CD dashboards

**Philosophy**: Information-dense but readable, keyboard-first, dark mode default.

```css
--color-bg: #0d1117;
--color-surface: #161b22;
--color-border: #30363d;
--color-text-primary: #e6edf3;
--color-text-secondary: #8b949e;
--color-accent: #58a6ff;
--color-accent-hover: #79c0ff;
--color-success: #3fb950;
--color-warning: #d29922;
--color-error: #f85149;
```

**Typography**: JetBrains Mono / Fira Code, 12/14/16/20/24px
**Corners**: 6px, no shadows, borders only
**Code**: 13px, line-height 1.6

```css

--color-surface: #ffffff;

--color-text-primary: #000000;
--color-text-secondary: #404040;
--color-accent: #000000;
--color-success: #000000;
--color-warning: #404040;
--color-error: #000000;
```

**Typography**: Times New Roman / Georgia (display), Arial (body), 12/16/20/32/48/72/96px
**Corners**: 0px (no rounding)
**Feature**: 2-4px solid borders, outlined cards, black/white only
