# 线框图符号指南 | Wireframe Symbols

## 核心目标

统一使用标准的线框图符号，让设计沟通更高效。

## 基础元素

### 文本

```markdown
标题文字
正文内容
[按钮]
```

### 容器

```markdown
┌──────────────┐
│              │
│              │
│              │
└──────────────┘
```

### 表单元素

```markdown
┌──────────────┐
│ [输入框    ] │
└──────────────┘

┌──────────────┐
│ [选择器   ▼] │
└──────────────┘

☑ 复选框
○ 单选框
```

### 按钮

```markdown
[主按钮]
[次要按钮]
[链接文字]
```

## 布局符号

### 页面结构

```markdown
+----------------------------------+
|  Header (导航栏)                  |
+----------------------------------+
|        |                         |
|  Side  |     Main Content        |
|  bar   |                         |
|        |                         |
+----------------------------------+
|  Footer                         |
+----------------------------------+
```

### 栅格

```markdown
|   |   |   |   |   |   |   |   |
+---+---+---+---+---+---+---+---+
|   |   |   |   |   |   |   |   |
+---+---+---+---+---+---+---+---+
```

## 交互符号

### 点击/触摸

```markdown
(○) 点击目标
```

### 手势

```markdown
→ swipe right (右滑)
← swipe left (左滑)
↑ swipe up (上滑)
↓ swipe down (下滑)
⊙ long press (长按)
```

### 状态变化

```markdown
A → B  (状态A变为状态B)
A + B  (A和B同时发生)
A | B  (A或B)
```


## 响应式断点

```markdown
Mobile:  < 768px
Tablet:  768px - 1024px
Desktop: > 1024px
```

## 颜色标注

```markdown
# 主色 - 主要操作
# 次色 - 次要操作
# 成功色 - 成功提示
# 警告色 - 警告提示
# 错误色 - 错误提示
# 灰色 - 禁用/次要信息
```

## 何时退出

- 所有页面都有线框图
- 组件状态已标注
- 交互流程已标注
- 响应式方案已说明
